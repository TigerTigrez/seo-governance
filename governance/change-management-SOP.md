# Change Management SOP (SEO-Impacting)
**Applies to:** redirects, robots/meta, canonical rules, nav, template head, URL patterns.

## Workflow
1) **Request** → PRD in /templates/prd-seo-change.md  
2) **Design** → Dev & SEO pair on approach; risk matrix completed  
3) **Review** → Security/Legal if applicable; Accessibility sign-off  
4) **Test** → Staging validation with /qa/prelaunch-SEO-smoke-test.md  
5) **Deploy** → Window with rollback plan; comms to stakeholders  
6) **Verify** → Post-deploy checks and 72-hour monitoring

## Evidence
- Before/after examples, test URLs, tool configs, log snippets (no PII)
