# Accessibility Checklist (WCAG 2.2 AA – SEO-relevant)
- [ ] Descriptive <title> and unique H1
- [ ] Headings form a logical outline (no skipped levels for layout)
- [ ] Links use meaningful text (avoid “click here”)
- [ ] Images: alt text or aria-hidden when decorative
- [ ] Form fields labeled; errors announced to screen readers
- [ ] Keyboard navigation works; visible focus states
- [ ] Color contrast ≥ AA for text/icons
- [ ] No SEO change reduces accessibility conformance
